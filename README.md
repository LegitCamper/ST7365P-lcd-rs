# ST7365P-lcd-rs

This is a Rust library for displays using the ST7365P driver with embedded_graphics, embedded_hal, and no_std, no_alloc support.

Two versions:
- Synchrouous (which is in master branch)
- Asyncrounous `branch = "async"`
